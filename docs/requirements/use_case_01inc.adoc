== Use-Case 01: Kandidaten verwalten
===	Kurzbeschreibung
Das System ermöglicht es Kandidaten anzulegen, zu bearbeiten oder zu entfernen.  

===	Kurzbeschreibung der Akteure
==== Admin
Ist für die Verwaltung zuständig

* Der Admin muss angemeldet sein
