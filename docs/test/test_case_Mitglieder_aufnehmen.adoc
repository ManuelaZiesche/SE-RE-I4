
= TC 02 - Mitglieder aufnehmen

*Vorbedingungen:*

- Kandidat(en), welche als Mitglied übernommen werden sollen, befinden sich bereits in der Datenbank
- Das Kandidatentab ist geöffnet und die Liste der Kandidaten ist sichtbar
